# Shape Painter


It's a paint demo with shape/layer by csharp winform.

https://www.github.com/surfsky/ShapePainter

![](./Doc/01.png)

# Features

- Shapes: line, oval, rect, rhombus, star, triangle, text
- Select
- Move
- Simply Undo/Redo
- Simply Serialize/Deserialize
- Save to png file
- Antialias
